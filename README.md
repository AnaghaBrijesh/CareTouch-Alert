#Emergency Touch Alert System

An assistive alert device built using **Particle Argon** during the **Omega overnight hackathon** at **Model Engineering College**. Designed for elderly and differently abled users, the system sends an **SMS alert to a caretaker** when the device is touched.

##Tech Stack
- C  
- Particle Argon  

##Key Features
- Touch-based emergency trigger  
- Instant SMS alert to caretaker  
- Simple and user-friendly design  
- Suitable for elderly and differently abled users  

##Purpose
Developed as a hackathon project to create an accessible and reliable emergency assistance solution.
